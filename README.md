# Multinode

A simple, distraction-free tool to organize your thoughts, tasks, or project notes into a nested tree structure.

---

## Getting Started

1.  **Create a Root**: Click **+ Root** to start your first main category.
2.  **Add Content**: Click inside the dark text box of any node to start typing.
3.  **Organize**: Use the arrow buttons or the structure buttons to arrange your notes into groups.

---

## Tool Groups Explained

### Global Visibility
*   **HEADERS**: Shows or hides the title bar for every node in your list.
*   **JUMPLINKS**: Shows or hides the quick-jump shortcut buttons on all nodes.

### Structure
*   **+ Root**: Adds a brand new, independent note at the bottom of the list.
*   **+ Sibling**: Adds a new note directly underneath the one you currently have selected.
*   **+ Child**: Adds a new note inside the selected note, creating a nested level.
*   **Collapse All**: Hides the content of every note, leaving only the headers or small boxes visible.
*   **Semi Collapse**: Hides the main text areas but keeps the headers visible for quick browsing.
*   **Uncollapse All**: Expands every note so you can see all your content at once.
*   **Reset Everything**: Completely clears your entire board (use with caution).

### Selected Node Actions
*   **Arrows (↑ ↓)**: Moves the selected note up or down within its current group.
*   **Arrows (← →)**: Moves a note out to a main level or pushes it inside the note above it.
*   **Dup ↑ / Dup ↓**: Creates an exact copy of the selected note either above or below it.
*   **Cut**: Prepares a note to be moved to a different spot.
*   **Copy**: Prepares a copy of the note to be placed elsewhere.
*   **Paste ↑ / Paste ↓**: Places your cut or copied note above or below the selection.
*   **Paste In**: Places your cut or copied note inside the selected note as a sub-item.
*   **Del Pull**: Deletes the selected note but keeps its "children" notes by moving them up a level.
*   **Del All**: Deletes the selected note and everything inside it.
*   **Collapse/Expand**: Shrinks or grows just the selected note.
*   **Toggle Wrap**: Switches the text area between a long single line and multiple lines.
*   **+ Add Jump**: Click this, then click another note to create a clickable shortcut link between them.

### Settings & Files
*   **Warnings**: When checked, the app will ask "Are you sure?" before deleting items.
*   **Universal Wrap**: Forces every note on the screen to either wrap text or stay on one line.
*   **Save JSON**: Downloads your entire board as a file to your computer.
*   **Load JSON**: Lets you pick a previously saved file to restore your notes.

---

## Realistic Uses

*   **Project Planning**: Use a Root node for a project name, "Child" nodes for specific tasks, and "Sibling" nodes for different phases.
*   **Coding Snippets**: Turn off "Wrap" to keep code lines clean and use "Jump" links to connect related functions.
*   **Brainstorming**: Quickly hit **+ Sibling** to dump ideas and use the arrow keys later to group them into categories.

---

## Tips & Shortcuts

*   **Selection**: Click any note to select it (it will turn blue).
*   **Multi-Select**: Hold the **Ctrl** key while clicking notes to select more than one.
*   **Deselect**: Click the empty background space or hit the **Esc** key to unselect everything.
*   **Quick View**: Hold **Alt + H** to quickly hide/show the header of a selected note.
*   **Quick Jumps**: Hold **Alt + J** to hide/show the jump links on a specific note.
